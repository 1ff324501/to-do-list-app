## Objetivo
Desenvolver um aplicativo simples para gerenciar tarefas.

## Funcionalidades
- Adicionar tarefas.
- Marcar tarefas como comcluídas.
- Excluir tarefas.

  ## Cronograma
  | Etapa               | Prazo      |
  |---------------------|------------|
  | Configuração inicial| Dia 1
  | Interface do usuário| Dia 2 E 3
  | Lógica do backend   | Dia 4
  | Teste e ajustes     | Dia 5
